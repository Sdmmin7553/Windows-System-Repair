## 🔍 Understanding the diagnostic tools

The application uses built-in Windows features to clean your system. Here is what happens behind the scenes when you trigger a scan.

**SFC Scan**
SFC stands for System File Checker. It looks at all protected system files on your computer. If it finds a modified file, it replaces it with the correct version from the Windows system folder. This fixes many random crashes and permission errors.

**DISM Repair**
DISM stands for Deployment Image Servicing and Management. It checks the health of your Windows image. If the base image contains errors, the tool downloads fresh files from Microsoft servers to restore your system to a healthy state.

**Runtime libraries**
Programs often need extra files to talk to your hardware. These files are called C++ Redistributables. If your computer lacks these, many games and apps will not start. Windows-System-Repair verifies if these exist and installs them if they are missing.
